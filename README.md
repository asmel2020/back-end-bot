### Instalação 🔧

_as seguintes etapas para executar o programa em um ambiente de desenvolvimento_

_install_

```
npm install
```

_build_

```
npm run build
```

_migrate banco do dado_

```
npx prisma migrate dev
```


_start_

```
npm run start OR npm run start:dev
```


_scan Qr_

```
http://localhost:8080/get-qr
```
